RawPhotoForge

RAW Photo Editor Written in Python

シンプルで高機能なRAW現像ソフトウェア。OpenCLとNumPyバックエンド対応で高速な画像処理エンジンにより、RAW写真をリアルタイムでプレビューしながら編集できます。明るさ、色相、彩度、輝度のトーンカーブで色や明るさを細かく調整可能。AIマスク（SAM）による部分補正、Lensfunによるレンズ補正、ExifToolによるメタデータ表示もサポート。

GitHub で見る 最新版ダウンロード

使い方

⚠️ 注意事項
本ソフトウェアはソースコード配布のみです。実行にはPython 3.11以上が必要です。
コマンド例はWindows用に書かれています。Linuxで使用する場合は適宜変更してください。
01

ソースコードを取得

GitHubのリリースページから自作Asset（zipファイル）をダウンロードしてください。

📥 最新版ダウンロード 📋 全リリースを見る
重要: GitHubの自動生成「Source code (zip/tar.gz)」ではなく、必ず自作Assetを使用してください。
ファイル名フォーマット: RawPhotoForge-v<バージョン>.zip

ダウンロード後、zipファイルを解凍し、任意の場所に配置します。

cd RawPhotoForge
02

AIモデルを配置

sam2.1-hiera-large ダウンロードページから以下のファイルを取得し、RawPhotoForgeフォルダ直下に配置します。

  • 📄 sam2.1_hiera_large.pt
  • 📄 sam2.1_hiera_l.yaml
配置例: RawPhotoForge\sam2.1_hiera_large.pt
03

ExifToolの導入

ExifTool 公式サイトからダウンロードして、環境変数PATHに追加してください。

確認手順:

exiftool -ver

バージョン番号が表示されればOKです。

04

依存ライブラリのインストール

pip install -r raw_photo_forge\requirements.txt

Pythonライブラリを一括でインストールします。エラーが出る場合は、Pythonのインストールやバージョンを確認してください。

05

RawPhotoForgeを起動

python raw_photo_forge\raw_photo_forge.py

ソフトが起動します。起動できない場合は、PythonやExifToolのPATH設定を確認してください。

始めよう

今すぐRawPhotoForgeをダウンロードして、RAW現像の新しい体験を始めましょう
